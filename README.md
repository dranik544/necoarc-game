# necoarc-game
cool neco arc game

RUS:

как запустить:

1. скачайте последний релиз
2. распакуйте .zip
3. выберите вашу ОС (Windows или Linux)
4. запустите

управление:

WASD - ходьба
SPACE - прыжок
SHIFT - бег
UP/DOWN - максимальная прорисовка
V - брюньяуу :3

ENG:

how to run:

1. download the latest release
2. unzip .zip
3. select your OS (Windows or Linux)
4. run

controls:

WASD - walk
SPACE - jump
SHIFT - run
UP/DOWN - maximum rendering
V - brunyauu :3
